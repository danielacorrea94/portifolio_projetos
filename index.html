Sobre mim

Meu nome é Daniela Leite Corrêa

Sou formada em Oceanografia (UERJ) e mestre em Sistemas Marinhos e Costeiros (UALG), Portugal. Trabalho com análise de dados científicos desde 2015, utilizando a linguação de programação MATLAB e migrando para Python posteriormente. Como oceanógrafa física, realizei análises de sensoriamento remoto e dados in situ, com séries temporais superiores a 30 anos e análises estatísticas de correlação e interpolação de dados. Devido a prévia experiência em linguagem de programação, realizei a migração de carreira para Cientista de Dados, onde aprimoro a cada dia minhas habilidades computacionais e amplio meus conhecimentos para além do Python.

Estou buscando uma oportunidade de trabalhar profissionalmente como Cientista de Dados para melhorar a tomada de decisão da empresa, através da construção de soluções usando dados. Como cientista gosto de trabalhar em grupo, gosto de receber sugestões, e tenho a ambição de trabalhar com novos dados, variáveis e plataformas.

Projetos em Ciência de Dados

Desenvolvimento de um Painel Gerencial para Marketplace de Entregas com o Streamlit

Nesse projeto, os conceitos de Programação em Python, manipulação de dados, pensamento estratégico e lógica de negócio, junto com ferramentas de desenvolvimetno web como o Streamlit e Github, foram usados para desenvolver um painel gerencial com as principais métricas de uma empresa marketplace de delivery de comida na India.

O resultado final do projeto foi um painel hospedado em um ambiente Cloud e disponibilizado através de um link web. O painel pode ser acessado por qualquer dispositivo conectado na internet.

Modelos de aprendizado de máquina para prever as vendas

O projeto foi desenvolvido com base no método de gerenciamento de projetos CRISP-DS (Cross-Industry Standard Process - Data Science, também conhecido como CRISP-DM), com as seguintes etapas:

  • Entendimento do Negócio
  • Coleta de dados
  • Limpeza de Dados
  • Análise Exploratória de Dados (EDA)
  • Preparação de dados
  • Modelagem e ajuste fino de Machine Learning
  • Modelo e Avaliação de Desempenho de Negócios/Resultados
  • Implantação do modelo
Um projeto end-to-end de Data Science com uma regressão adaptada para séries temporais como solução criou quatro modelos de aprendizado de máquina para prever as vendas. As previsões podem ser acessadas pelos usuários por meio de um bot do aplicativo de smartphone Telegram.

As ferramentas utilizadas foram:

  • Python, Pandas, Numpy e Seaborn.
  • Anaconda, Pycharm e Jupyter Notebook.
  • Mapas interativos com Plotly e Folium.
  • Random Forest Regressor
  • XGBoost Regressor
  • Time-Series method
